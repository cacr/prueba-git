Estado del Arte.

El estado del arte nos da a conocer lo que existe hoy en día con respecto a algo en particular, en este caso, códigos QR su lectura, utilización y creación, cuáles son las técnicas de desarrollo de software, ya que esta tesis consta principalmente del desarrollo de uno, nuevo carnet de identidad chileno, la base de datos que se deberá utilizar.
Código QR: hoy en día este tipo de código se está utilizando mucho, ya que fue diseñado con el fin de ser rápido y para que contenga una alta concentración de carateres, hoy en día es utilizado para visitar una web, añadir datos a una agenda, reconocer algún producto, acceder a un texto, un documento, mapas, coordenadas, para conseguir información, etc. En general se les ha dado y se les están dando muchas utilizaciones.
Software Ágil: es lo que está siendo utilizado últimamente, ya que se está valorando mucho el tiempo hoy en día, porque todo es muy rápido y cambiante a cada instante, por lo que se necesitan resultados más luego que antes y este tipo de metodología fue creada para eso y se está potenciando mucho hoy en día.
Base de Datos: hoy en día existen muchos tipos de bases de datos y se siguen desarrollando metodologías mejores y más completas para un mejor funcionamiento de los sistemas. Estas son utilizadas en prácticamente todos los sistemas existentes, tanto en línea como locales. En el caso de este sistema, se debe contar con una tipo de base de datos que capaz de soportar una alta cantidad de datos, ya que será muy robusta.
Lectores de Códigos QR: existe un sinfín de aparatos que reconocen este tipo de códigos, desde celulares hasta dispositivos de gran acabado, inclusive en algunos de ellos es posible cargar sistemas a base de lenguajes de programación.

Metodologías, Técnicas, Hipótesis, Tecnologías.

La metodología en particular que se utilizará será la del reconocimiento de código QR mediante dispositivo especializado para este, que permita identificar a una persona mediante el código del carnet de identidad. Se utilizarán aparatos tecnológicos que reconozcan código QR, que además soporten lenguajes de programación que servirá para desarrollar un sistema específico para una función en particular. Este sistema debería ser capaz en teoría poder realizar un completo registro de las asistencias de cada alumno de plan común y que esta quede almacenada en una base de datos permanente, con el fin de darle la mayor utilización posible aparte del registro en sí.
